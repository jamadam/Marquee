=encoding utf8

=head1 NAME

Marquee::SSIHandler - SSIハンドラーベースクラス

=head1 SYNOPSIS

    package Marquee::SSIHandler::EPL;
    use Mojo::Base 'Marquee::SSIHandler';
    
    sub render {
        my ($self, $path) = @_;
        
        ...;
        
        return $out;
    }

=head1 DESCRIPTION

本クラスは、ハンドラークラスに継承されるベースクラスです。サブクラスはC<render>を実装しなければなりません。

=head1 ATTRIBUTES

L<Marquee::SSIHandler>は下記の属性を実装します。

=head2 app

Marqueeアプリインスタンスを格納します。.

    $ep->app($app);

=head2 log

C<Mojo::Log>インスタンスです。デフォルトは、もし存在すればC<$app-E<gt>log>です。

=head1 CLASS METHODS

L<Marquee::SSIHandler>は下記のクラスメソッドを実装します。

=head2 Class->new;

コンストラクター。

=head2 Class->current_template;

現在のテンプレートを再帰的に検出します。

    my $current_template = Marquee::SSIHandler->current_template;
    my $parent_template = Marquee::SSIHandler->current_template(1);

=head1 INSTANCE METHODS

L<Marquee::SSIHandler>は下記のインスタンスメソッドを実装します。

=head2 $instance->render;

テンプレートをレンダリングします。サブクラスはこのメソッドをオーバーライドしなければなりません。
    
    sub render {
        my ($self, $path) = @_;
        
        ...;
        
        return $out;
    }

=head2 $instance->render_traceable;

テンプレート名を再帰的にスタックすることで、トレース可能にレンダリングします。

=cut
